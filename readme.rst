Deskripsi singkat

Web Yogasana adalah sebuah web bertujuan untuk mendaftar yoga dan bertujuan untuk memudahkan
penggunanya dalam mencari informasi mengenai yoga.

Tools

Untuk Frontend gunakan Javascript, HTML, CSS dan menggunakan Framework Code Igniter 3
Untuk Backend gunakan Javascript, HTML, CSS, dan menggunakan Framework CodeIgniter 3

 Dashboard page

Frontend :
 Landing Page/Home Page
 About page
 Classes Page
 yoga page

 Contact page
